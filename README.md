# Sales Dashboard

## TL;DR
Interactive Power BI dashboard visualizing monthly and product-wise sales trends, KPIs, and performance metrics to support business decisions.

## Tools
Power BI, Excel

## Dataset
Sales records: Date, Product, Region, Revenue, Quantity, Customer details

## Steps
1. Data Cleaning: Removed duplicates, formatted dates, and standardized categories.
2. Data Modeling: Built relationships between tables (Sales, Customers, Products).
3. Dashboard Development: Added KPIs, slicers, drill-through pages, and trend charts.
4. Insights: Analyzed top-performing products, regions, and monthly trends.

## Key Results
- Enabled sales team to track performance in real-time.
- Improved reporting efficiency by 30%.
- Highlighted top-selling products and regions.

## How to Use
- Open `Sales_Dashboard.pbix` in Power BI Desktop.
- Use slicers to filter by region, product, or month.
- Explore drill-through pages for detailed insights.
